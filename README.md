## Các chức năng chính

### Chức năng Admin:
1. Đăng nhập
2. Quản lý thí sinh
3. Tạo kì thi
4. Quản lý kì thi

6. Quản lý Admin

2. Chỉnh sửa thông tin
3. Xem lịch sử thi
4. Thi
